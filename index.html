2. Matriz de Influencia Directa (MID)

La matriz cuantifica cómo cada proceso impacta a los demás. Una puntuación de 0 indica nula influencia, mientras que 3 representa una influencia fuerte. Las filas representan la capacidad de influir de una variable, y las columnas su nivel de dependencia de otras. Pase el cursor sobre las celdas para ver las conexiones.

0 - Nula 1 - Débil 2 - Media 3 - Fuerte

3. Plano de Influencia y Dependencia

Este es el corazón del análisis MICMAC. Mapea las variables según su puntuación total de Influencia (Eje Y) y Dependencia (Eje X). El plano se divide en cuatro cuadrantes estratégicos que revelan el verdadero rol de cada proceso en la dinámica de Nike Jordan. Haga clic o pase el cursor sobre los puntos para identificar los procesos.

🔥 I. Variables de Enlace

Alta Influencia / Alta Dependencia. Son altamente inestables. Cualquier cambio aquí repercute en todo el sistema (Ej. Innovación M1).

👑 II. Variables Determinantes

Alta Influencia / Baja Dependencia. Las variables "maestras". Son los motores que mueven a Nike (Ej. Branding E4).

🎯 III. Variables de Resultado

Baja Influencia / Alta Dependencia. Son los efectos o indicadores de éxito del sistema (Ej. Ventas M2).

⚙️ IV. Variables Autónomas

Baja Influencia / Baja Dependencia. Actúan con relativa independencia y no detienen el core del negocio (Ej. RRHH A2).

4. Conclusiones Estratégicas

Basado en el posicionamiento de las variables en el plano MICMAC, se derivan tres conclusiones fundamentales que explican el éxito sostenido de la marca y señalan sus áreas críticas de vulnerabilidad. Explore las tarjetas interactivas.

⭐

La Variable Clave es el Branding

El proceso con mayor influencia del sistema (E4).

En el caso Nike Jordan, el valor no reside solo en el zapato, sino en lo que la marca representa. Es la variable con mayor influencia directa sobre las ventas y el marketing. Su posicionamiento en el cuadrante Determinante ratifica que es el motor principal.

Dependencia Digital

La infraestructura como base del negocio moderno.

La Expansión Digital (E2) y las TI (A3) forman un bloque crítico. Nike ha pasado de ser una empresa tradicional de productos a una plataforma de servicios digitales ("Direct to Consumer"), convirtiendo a la tecnología en una variable determinante para la supervivencia.

El Conflicto de la Innovación

El punto de equilibrio más inestable (E3).

La Innovación (E3) está en el centro del mapa de Enlace. Es el motor del deseo, pero si Nike deja de innovar, el Branding se erosiona y las ventas caen drásticamente. Requiere alimentación constante de inversión y talento para no frenar el ciclo.
